# 按鍵腳位修正完成

## 修正內容

已將按鍵 A 的 GPIO 腳位從 PA.0 修正為 PB.15，符合實際硬體設計。

## 修改項目

### 1. 主程式配置 (main.c)
```c
// 修改前
{PA, BIT0, true, 50, 3000, 1000} // PA.0

// 修改後  
{PB, BIT15, true, 50, 3000, 1000} // PB.15
```

## 硬體配置確認

### GPIO 配置
- **腳位**: PB.15
- **模式**: 輸入模式
- **觸發**: 低電位觸發 (active_low = true)
- **內建電阻**: 上拉電阻啟用

### 按鍵連接方式
```
PB.15 ──┬── 按鍵開關 ── GND
        │
       10kΩ (內建上拉)
        │
       3.3V
```

## 使用說明

按鍵 A (PB.15) 現在可以正常使用：

1. **短按**: 顯示狀態訊息
2. **長按 3 秒**: 循環切換 LED 顯示模式
3. **雙擊**: 在正常模式與快速模式間切換

所有按鍵操作都會透過 UART0 (115200 baud) 輸出詳細的狀態資訊。
