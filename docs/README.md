# Documentation index

這裡收納專案的說明文件與設計/實作筆記，已將各範例與子專案的 Markdown 檔集中在 `docs/` 下的對應子目錄。

主要目錄快速連結：

- `docs/IMPLEMENTATION_NOTES.md` — REPL (BLE AT) 與近期實作細節紀錄
- `docs/SampleCode/RL_SPORT/` — RL_SPORT 範例文件（包含 BLE AT 指令設計）
- `docs/SampleCode/Smart_Box/` — Smart_Box 範例文件（Modbus / Mesh / BLE）

如何使用：

1. 在本地檢視：打開 `docs/` 下對應檔案以閱讀詳細說明。
2. 若要回溯原始程式碼：前往 `SampleCode/<project>/` 尋找示範程式與範例實作。

如果需要，我可以產生一份更詳細的 HTML 索引或自動化腳本（例如 mkdocs）來發布文件站點。
